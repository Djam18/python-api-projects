# Python API Projects

Flask API projects from roadmap.sh - learning REST APIs.

## Projects

| Project | Level | Stack | Description |
|---------|-------|-------|-------------|
| weather-api | intermediate | Flask + Redis | Weather API with caching |
| todo-api | beginner | Flask + SQLite | Todo CRUD API |
| blogging-api | intermediate | Flask + JWT | Blog with auth |
| expense-api | beginner | Flask + SQLite | Expense tracker |
| url-shortener | intermediate | Flask + SQLite | URL shortener with analytics |
| markdown-notes | beginner | Flask + markdown2 | Notes with markdown |
| caching-proxy | intermediate | Flask + Redis | HTTP caching proxy |
| workout-tracker | beginner | Flask + SQLite | Workout progress tracker |
| broadcast-server | advanced | asyncio + websockets | WebSocket chat |

## Setup
pip install -r requirements.txt

## Run any project
cd <project-name>
python app.py
